# Riemannian Proximal Bundle Method

Official implementation of **"Convergence Rates for Riemannian Proximal Bundle Methods"**.

## Requirements

* **Julia:** Julia 1.9+
* **Packages:** All dependencies are listed in `Project.toml`. To install, launch Julia in the repository root and run:
  ```julia
  using Pkg
  Pkg.instantiate()
  ```
* **Jupyter (optional):** To run the interactive notebooks, install IJulia:
  ```julia
  using Pkg
  Pkg.add("IJulia")
  ```

## Experiments

### Riemannian Median on SPD Matrices

Computes the Fréchet median of $N$ random Symmetric Positive Definite matrices and compares RPB against RCBM, PBA, and SGM (and their first-order retraction variants).

Key parameters (configurable at the top of `experiment_median_SPD.jl` or in `experiments/median_SPD.ipynb`):

| Parameter | Default | Description |
|-----------|---------|-------------|
| `N` | 20 | Number of data points |
| `spd_dims` | [3, 5, 15, 30, 55] | SPD matrix dimensions to benchmark |
| `atol` | 1e-12 | Convergence tolerance |
| `maxiter` | 1000 | Maximum iterations per method |
| `seed_argument` | 57 | RNG seed |
| `PHASE1_MULTIPLIER` | 2 | Phase 1 runs for `PHASE1_MULTIPLIER × maxiter` iterations to estimate the true minimum |
| `PHASE2_GAP_TOL` | 1e-12 | Phase 2 stops when objective gap drops below this threshold |

Run the full experiment:
```bash
julia experiment_median_SPD.jl
```

Or open the interactive notebook: `experiments/median_SPD.ipynb`

Plots are generated by:
```bash
julia plot_median_SPD.jl
```

### TV Denoising on the Hyperbolic Manifold

Solves a total variation denoising problem on the power manifold $(\mathbb{H}^2)^n$ and compares RPB against RCBM, PBA, and SGM.

Key parameters (configurable at the top of `experiment_denoising_hyperbolic.jl` or in `experiments/denoising_hyperbolic.ipynb`):

| Parameter | Default | Description |
|-----------|---------|-------------|
| `num_points` | 496 | Signal discretization |
| `sigma` | 0.3 | Noise level (σ of Riemannian Gaussian) |
| `alpha` | 0.5 | TV regularization strength |
| `atol` | 1e-8 | Convergence tolerance |
| `max_iter` | 100,000 | Maximum RPB iterations |
| `proximal_parameter` | 1/num_points | Initial proximal parameter $\rho_0$ |
| `trust_parameter` | 0.001 | Descent condition threshold $\beta$ |
| `retraction_error` | 1.0 | Retraction error constant |
| `transport_error` | 1.0 | Transport error constant |
| `back_tracking_fact` | 2 | Proximal parameter backtracking factor |
| `PHASE1_MULTIPLIER` | 1 | Phase 1 multiplier for estimating the true minimum |
| `PHASE2_GAP_TOL` | 1e-8 | Phase 2 gap tolerance |

Run the full experiment:
```bash
julia experiment_denoising_hyperbolic.jl
```

Or open the interactive notebook: `experiments/denoising_hyperbolic.ipynb`

Plots are generated by:
```bash
julia plot_denoising_hyperbolic.jl
```
